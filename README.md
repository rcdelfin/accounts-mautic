meteor-accounts-mautic
============================

A meteor package for LinkedIn's login service.

Package Dependencies
----------------------

* accounts-base
* accounts-oauth
* rcdelfin:mautic

Install
-----------
```
meteor add rcdelfin:accounts-mautic
```
